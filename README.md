
# QuizGame  

# Features:  

- Displays multiple-choice questions  

- Accepts user input  

- Tracks correct answers  

- Displays final score

 # Technologies Used  
 
- Core Java (OOP)
  
- ArrayList for storing questions

- Scanner for input

- Simple logic & conditionals
